# disrust

**Replaces:** Discord desktop client (TUI alternative) | **Language:** 🦀 Rust | **Install:** `cargo install disrust`

## Purpose
Terminal-based Discord client.

## Launch
```
disrust
```

## Key bindings
| Key | Action |
|-----|--------|
| `↑`/`↓` | Navigate |
| `Tab` | Switch panes |
| `Enter` | Select / send |
| `Ctrl+Q` | Quit |

## Gotchas
- Unofficial — using third-party Discord clients can violate Discord's ToS; use at your own risk.
- Token-based login requires manually extracting your Discord token.
