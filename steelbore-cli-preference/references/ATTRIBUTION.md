# Attribution

The reference files in this directory are original writing by the
`steelbore-cli-preference` skill authors. Command flags, default behaviour,
and example patterns were cross-checked against the following upstream
sources, in decreasing order of reliance:

1. **[tldr-pages](https://github.com/tldr-pages/tldr)** — community-maintained
   simplified man pages, licensed under **CC BY 4.0**.
   Where a tldr page existed for a tool, it was used as a factual
   cross-check for flags and invocation patterns. No verbatim tldr text
   appears in these references — all prose is rewritten in agent-structured
   form.

2. **Each tool's official README and `--help` output** — for flags and
   behaviour not covered by tldr.

3. **Manual pages (`man`, `info`)** — for standards-track tools like
   `curl`, `ffmpeg`, `sudo`, `flatpak`, `nix`.
